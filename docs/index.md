目标
===========

宝贝回家 的主要宗旨是为帮助寻找失踪儿童及一些流浪乞讨的孩子找家，为孩子家长及志愿者提供一个信息沟通的平台。

我们希望通过技术的手段，改善公益的现状，为大家做好技术保障。比如近期做的小程序，希望可以方便大家进行交流、沟通。
